# Un viaje al corazón de la JVM

Este repositorio tiene como propósito servir de apoyo a la presentación **"Un viaje al corazón de la JVM"**. Aquí encontrarás:

- La presentación en formato PowerPoint.
- Las demos utilizadas durante la charla.

El objetivo es explorar el funcionamiento interno de la Máquina Virtual de Java (JVM), sus componentes principales y cómo ejecuta el código Java.

## Recursos adicionales

Para experimentar y profundizar de forma interactiva en los conceptos presentados, puedes visitar el siguiente playground:
